Prerequisites

1- Update the packages

sudo apt update && sudo apt upgrade

2- Create a new user

Create a new user for django & give sudo permissions:
adduser <djangouser>
Where djangouser is the arbitary username for the django application
usermod -aG sudo <djangouser>

3- Secure the linux server

Set host name for the linux server
sudo hostnamectl set-hostname <linux_server_name>
sudo vim /etc/hosts
add the following text in the file and save it (:wq)
<linux_server_ip> <linux-server-name>
where <linux_server_ip> is the ip address of linode/ digitalocean...

4- Create secure ssh connection to the linux server

On local machine

Generate the ssh keys on your local machine using the following commands:
ssh-keygen -b 4096
Accept the defaults by pressing enter and when the process is done, you will have
two files in ~/.ssh directory names id_rsa and id_rsa.pub
Secure the ssh login to the linux server and disable the password login:

On linux server

mkdir -p ~/.ssh
Add the public key generated on the local machine in the following file:
vim ~/.ssh/authorized_keys
chmod -R go= ~/.ssh
sudo chmod 700 ~/.ssh/
sudo chmod 600 ~/.ssh/*
Disable root login and password based login
sudo vim /etc/ssh/sshd_config
Modify the following parameters in the file:
"PermitRootLogin no"
"PasswordAuthentication no"
sudo systemctl restart sshd
